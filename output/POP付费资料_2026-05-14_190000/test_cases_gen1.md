## F1: POP资料库-列表查看

### TC-001: 正常搜索查看资料列表
- **前置条件**: 用户已登录教务后台，且系统中存在POP资料数据
- **测试步骤**:
  1. 进入教务后台 → POP资料库
  2. 在搜索框输入资料名称关键词（如"执业医师"）
  3. 点击搜索按钮
- **预期结果**: 列表正确显示匹配关键词的资料，显示字段包括：资料ID、资料名称、资料类型、所属科目、是否上架、添加时间、更新时间
- **优先级**: P0
- **测试类型**: 功能

### TC-002: 按科目筛选资料
- **前置条件**: 系统中存在多个科目的POP资料
- **测试步骤**:
  1. 进入教务后台 → POP资料库
  2. 在筛选条件中选择科目（如"中医基础"）
  3. 点击筛选/搜索
- **预期结果**: 列表仅显示所选科目的资料，其他字段显示正确
- **优先级**: P1
- **测试类型**: 功能

### TC-003: 按上架状态下拉筛选
- **前置条件**: 系统中同时存在上架和未上架的资料
- **测试步骤**:
  1. 进入教务后台 → POP资料库
  2. 选择上架状态下拉筛选（如"已上架"）
  3. 点击筛选
- **预期结果**: 列表仅显示符合筛选条件的资料，可正常翻页
- **优先级**: P1
- **测试类型**: 边界

### TC-004: 搜索无结果展示
- **前置条件**: 系统中有POP资料数据
- **测试步骤**:
  1. 进入教务后台 → POP资料库
  2. 输入系统中不存在的资料名称关键词（如"xyznotexist123"）
  3. 点击搜索
- **预期结果**: 显示空状态或"未找到相关资料"提示，列表为空
- **优先级**: P2
- **测试类型**: 异常

## F2: POP资料库-添加资料

### TC-005: 正常添加资料
- **前置条件**: 用户已登录教务后台
- **测试步骤**:
  1. 进入教务后台 → POP资料库
  2. 点击"添加资料"按钮
  3. 填写资料信息：资料名称（唯一）、资料类型、所属科目、是否上架
  4. 点击"保存"按钮
- **预期结果**: 弹出成功提示，资料自动出现在列表第一行或按时间排序位置
- **优先级**: P0
- **测试类型**: 功能

### TC-006: 添加资料-必填字段为空
- **前置条件**: 用户已登录教务后台
- **测试步骤**:
  1. 进入教务后台 → POP资料库
  2. 点击"添加资料"按钮
  3. 不填写资料名称，直接点击"保存"
- **预期结果**: 表单校验提示"资料名称为必填项"，不允许提交
- **优先级**: P0
- **测试类型**: 异常

### TC-007: 添加资料-重复资料名称
- **前置条件**: 系统中已存在名为"执业医师模拟题"的资料
- **测试步骤**:
  1. 进入教务后台 → POP资料库
  2. 点击"添加资料"按钮
  3. 填写资料名称为"执业医师模拟题"（重复）
  4. 填写其他必填信息
  5. 点击"保存"
- **预期结果**: 提示"资料名称已存在，请更换"，保存失败
- **优先级**: P1
- **测试类型**: 异常

### TC-008: 添加资料-资料名称超长边界
- **前置条件**: 用户已登录教务后台
- **测试步骤**:
  1. 进入教务后台 → POP资料库
  2. 点击"添加资料"按钮
  3. 填写资料名称为超长字符串（超过100字符）
  4. 填写其他必填信息
  5. 点击"保存"
- **预期结果**: 提示"资料名称不能超过100字符"，或前端截断处理
- **优先级**: P2
- **测试类型**: 边界

## F3: POP资料库-编辑资料

### TC-009: 正常编辑资料
- **前置条件**: 系统中存在可编辑的POP资料
- **测试步骤**:
  1. 进入教务后台 → POP资料库
  2. 找到目标资料，点击"编辑"按钮
  3. 修改资料名称（如改为"执业医师模拟题v2"）
  4. 点击"保存"
- **预期结果**: 弹出成功提示，列表中该资料的名称和更新时间已更新
- **优先级**: P0
- **测试类型**: 功能

### TC-010: 编辑资料-修改上架状态
- **前置条件**: 系统中存在已上架的资料
- **测试步骤**:
  1. 进入教务后台 → POP资料库
  2. 找到目标资料，点击"编辑"
  3. 将"是否上架"从"是"改为"否"
  4. 点击"保存"
- **预期结果**: 资料上架状态变更，列表中显示状态更新
- **优先级**: P1
- **测试类型**: 边界

### TC-011: 正常软删除资料
- **前置条件**: 系统中存在可删除的POP资料
- **测试步骤**:

  2. 找到目标资料，点击"删除"按钮
  3. 确认删除操作
- **预期结果**: 资料从列表中移除，软删除成功（数据库标记删除）
- **优先级**: P1
- **测试类型**: 功能

### TC-012: 删除资料-取消删除操作
- **前置条件**: 系统中存在可删除的POP资料
- **测试步骤**:
  1. 进入教务后台 → POP资料库
  2. 找到目标资料，点击"删除"按钮
  3. 在弹窗中点击"取消"
- **预期结果**: 弹窗关闭，资料未被删除，列表保持不变
- **优先级**: P2

## F5: POP资料库-批量操作

### TC-013: 批量添加资料
- **前置条件**: 用户已登录教务后台，有批量导入模板
- **测试步骤**:
  1. 进入教务后台 → POP资料库
  2. 点击"批量添加"按钮
  3. 上传符合格式的批量资料文件（Excel/CSV）
  4. 点击确认导入
- **预期结果**: 显示导入进度，导入成功后列表刷新显示新增资料
- **优先级**: P0
- **测试类型**: 功能

### TC-014: 批量添加-文件格式错误
- **前置条件**: 用户已登录教务后台
- **测试步骤**:
  1. 进入教务后台 → POP资料库
  2. 点击"批量添加"
  3. 上传格式错误的文件（如Word文档）
  4. 点击确认导入
- **预期结果**: 提示"文件格式不正确，请上传Excel或CSV文件"
- **优先级**: P1
- **测试类型**: 异常

### TC-015: 批量删除资料
- **前置条件**: 系统中存在多个POP资料
- **测试步骤**:
  1. 进入教务后台 → POP资料库
  2. 勾选2个或以上资料前的复选框
  3. 点击"批量删除"按钮
  4. 确认删除操作
- **预期结果**: 所选资料全部从列表中移除
- **优先级**: P1
- **测试类型**: 功能

### TC-016: 批量删除-未选择任何资料
- **前置条件**: 系统中存在POP资料
- **测试步骤**:
  1. 进入教务后台 → POP资料库
  2. 不勾选任何资料
  3. 点击"批量删除"按钮
- **预期结果**: 提示"请先选择要删除的资料"或按钮置灰不可点击
- **优先级**: P2
- **测试类型**: 异常
