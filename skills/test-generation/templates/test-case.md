# 测试用例模板

## 格式规范

```markdown
### TC-{ID}: {测试名称}
- **前置条件**: {测试前需满足的条件}
- **测试步骤**:
  1. {步骤1}
  2. {步骤2}
- **预期结果**: {预期的输出或行为}
- **优先级**: {P0/P1/P2}
- **测试类型**: {功能/边界/异常}
```

## 完整示例

```markdown
### TC-001: 用户注册-用户名格式验证
- **前置条件**: 无
- **测试步骤**:
  1. 输入用户名"abc"（3字符）
  2. 输入合规密码"Pass1234"和邮箱"test@example.com"
  3. 点击注册按钮
- **预期结果**: 注册失败，提示"用户名长度需6-20字符"
- **优先级**: P0
- **测试类型**: 边界
```
